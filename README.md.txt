# AI Toolkit - 智能对话与创作平台

基于 OpenAI 兼容 API 的多功能 AI 工具箱，支持对话、情感分析、摘要、图像生成，并完整追踪 Token 消耗。**专为 MIMO 7亿 Token 申请设计**，展示完整的 AI 应用开发能力。

## 🚀 功能特性

- 💬 **智能对话**：多轮上下文对话，支持主流大语言模型
- 🎭 **情感分析**：快速识别文本情绪倾向
- 📄 **文本摘要**：长文本核心内容提取
- 🎨 **图像生成**：通过描述生成图片 (DALL·E 兼容)
- 📊 **Token 用量统计**：实时追踪 API 消耗，总量累计展示
- 🌓 **深色模式**：舒适的夜间使用体验
- 💾 **本地配置**：API Key 等保存于浏览器，安全可控

## 🛠️ 技术栈

- **后端**: Node.js + Express
- **前端**: 原生 JS + Tailwind CSS
- **API**: 标准 OpenAI Chat Completions & Images Generations 接口
